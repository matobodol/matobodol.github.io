Installing arch linux and make it useble

Pemasangan base arch linux

Menghubungkan ke internet

  1. Via iwctl
    • # iwctl
    • [iwctl]# device list
    • [iwctl]# station device scan
    • [iwctl]# station device get-networks
    • [iwctl]# station devise connect SSID
  2. Via usb tethering
    • Hubungkan hape android ke usb
    • aktifkan usb tethering di pengaturan hape

Perbarui jam sistem

  • timedatectl set-ntp true

Mengatur partisi

  • cfdisk /dev/sda

Format partisi

  • mkfs.ext4 /dev/partition_root
  • mkfs.ext4 /dev/partition_home
  • mkswap /dev/partition_swap

Mount partisi

  • mount /dev/partisi_root /mnt
  • mount /dev/partisi_home /mnt/home
  • swapon /dev/partisi_swap

Install base

  • pacman -S base linux linux-firmware base base-devel linux-firmware nano

Konfigurasirasi sistem

Fstab

  • genfstab -U /mnt >> /mnt/etc/fstab

Chroot

  • arch-chroot /mnt

Zona waktu

  • ln -sf /usr/share/zoneinfo/Region/city /etc/localtime
  • hwclock --systohc

Lokalisasi

  • nano /etc/locale.gen hapus komentar pada en_US.UTF-8
  • locale-gen
  • nano /etc/locale.conf isi dengan LANG=en_US.UTF-8 lalu simpan

Konfigurasi jaringan

  • nano /etc/hostname isi dengan myarch
  • nano /etc/hosts masukan seperti dibawah:
    •   127.0.0.1   localhost
  ::1         localhost
  127.0.1.1   myarch.localdomain   myarch

Membuat sandi root

  • passwd

Boot loader

  • pacman -S grub networkmanager dialog wireless_tools wpa_supplicant os-prober mtools dosfstools gvfs gvfs-mtp ntfs-3g
  • grub-install --target=i386-pc /dev/sda
  • grub-mkconfig -o /boot/grub/grub.cfg
  • exit
  • umount -a
  • reboot

Login sebagai root

Aktifkan koneksi internet

  1. Via networkmanager
    • systemctl start NetworkManager
    • nmtui --> aktivate a connection --> hubungkan ke wifi --> masukan sandi
    • systemctl enable NetworkManager
  2. Via usb tethering
    • hubungkan hape android ke usb
    • aktifkan usb tethering di pengaturan hape

Membuat user baru

  • useradd -m -G wheel myname
  • passwd myname

Install kde plasma minimal

Install display manager

  • pacman -S sddm sddm-kcm
  • systemctl enable sddm

Install desktop kde plasma

  • pacman -S plasma-desktop konsole dolphin spectacle ark gwenview elisa discover powerdevil plasma-systemmonitor kinfocenter khelpcenter
  • pacman -S dolphin-plugins plasma-pa plasma-nm kdeplasma-addons kde-system-meta appstream appstream-qt packagekit packagekit-qt5 gst-libav

Mengubah tema kursor sddm

  • buka System Settings -->Startup and Shutdown -->Login Screen (SDDM)
  • pilih tema breeze, klik Sinchcronize Settings.. dan terahir klik apply
  • reboot

Installing arch linux and make it useable

Tutorial ini ditulis sebagai dokumentasi pribadi dalam menginstal arch linux dan beberapa setup untuk membuatnya menjadi lebih mudah digunakan. Meskipun demikian, tutorial ini semoga membantu dalam menginstal arch linux.

Sebelum memulai,
berikut adalah penjelasan tentang setup yang saya pakai:
  • proses instal dengan sistem bios/mbr
  • membuat partisi /home sendiri
  • dual boot windows dan arch yang sekarang sedang akan di instal
  • pembagian partisi menggunakan gparted live xubuntu
  • menggunakan desktop kde plasma (minimal)

Pemasangan base linux

  1. Menghubungkan ke internet

    1. Via iwctl

      • # iwctl
      • [iwd]# device list
      • [iwd]# station device scan
      • [iwd]# station device get-connect
      • [iwd]# station device connect SSID

    2. Via usb tethering

      • Hubungkan android ke usb
      • Aktifkan usb tethering di pengaturan android

  2. Perbarui jam sistem

    • # timedatectl set-ntp true

  3. Mengatur partisi

    • # cfdisk /dev/sda

  4. Mount partisi

    • # mount /dev/partisi_root /mnt
    • # mkdir /mnt/home
    • # mount /dev/partisi_home /mnt/home
    • # swapon /dev/partisi_swap

  5. Format partisi

    • # mkfs.ext4 /dev/partisi_root
    • # mkfs.ext4 /dev/partisi_home
    • # mkswap /dev/partisi_swap

  6. Instal base sistem

    • pacstrap /mnt base base-devel linux linux-firmware linux-headers nano

Konfigurasi sistem

  1. Fstab

    • # genfstab -U >> /mnt/etc/fstab

  2. Chroot

    • # arh-chroot /mnt

  3. Zona waktu

    • # ln -sf /usr/share/zoneinfo/Region/City /etc/localtime
    • # hwclock --systohc

  4. Lokalisasi

    • # nano /etc/locale.gen
      • cari dan hapus komentar (tanda #) pada baris en_US.UTF-8
    • #locale-gen
    • #nano /etc/locale.conf
      • edit dan isi: LANG=en_UST.UTF-8

  5. Konfigurasi Jaringan

    • # nano /etc/hostname (ketik MyArch lalu simpan)
    • # nano /etc/hosts (lalu isi sebagai berikut)
    • 127.0.0.1localhost
      ::1localhost
      127.0.0.1MyArch.localdomainMyArch

  6. Membuat sandi untuk root

    • # passwd

  7. Boot loader

    • # pacman -S grub networkmanager dialog wireless_tools wpa_supplicant os-prober mtools dosfstools gvfs gvfs-mtp ntfs-3g
    • # grub-install --targer=i386-pc /dev/sda
    • # grub-mkconfig -o /boot/grub/grub.cfg
    • # exit
    • # umount -a
    • # reboot

  8. Login sebagai root....

  9. Aktifkan koneksi jaringan

    1. Menghubungkan ke wifi

      • # systemctl start NetworkManager
      • # nmtui → activate a connection → hubungkan ke wifi, masukan sandi.
      • # systemctl enable NetworkManager

    2. Via usb tethering

      • Hubungkan android ke usb
      • Aktifkan usb tethering di pengaturan android

  10. Membuat user baru

    • # useradd -m -G wheel MyName
    • # passwd MyName
    • # EDITOR=nano visudo
    • cari dan hapus tada # pada baris %wheel ALL=(ALL) ALL
      • lalu simpan

Install desktop kde plasma minimal

  1. Instal display manager

    • # pacman -S sddm sddm-kcm
    • # systemctl enable sdm

  2. Instal kde plasma

    • # pacman -S plasma-desktop konsole dolphin spectacle ark gwenview plasma-systemmonitor kinfocenter discover powerdevil
    • # pacman -S dolphin-plugins packagekit packagekit-qt5 appstream appstream-qt plasma-nm plasma-pa kdeplasma-addons kde-system-meta khelpcenter gst-libav

  3. mengubah tema dan kursor sddm

    • buka System Settings → Startup and Shutdown → Login Screen (SDDM)
    • pilih tema breeze, klik Sinchronize Settings.. dan terahir klik apply
    • Reboot..

  4. List service systemd

    • # nano /etc/systemd/system.conf
    • hapus tanda # di baris ShowStatus=yes
